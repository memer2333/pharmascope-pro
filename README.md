# 💊 PharmaScope Pro
### Clinical-Grade Drug Information for Healthcare Professionals

PharmaScope Pro is a high-performance web application designed to provide structured, evidence-based medical data at the point of care. It aggregates data from authoritative APIs (FDA, NIH RxNav, DailyMed) and utilizes locally curated datasets for specialized clinical tools.

## ✨ Key Features
- **Comprehensive Drug Profiles:** Search over 100K drug labels for indications, pharmacology, safety warnings, and contraindications.
- **Clinical Calculators:** Built-in Creatinine Clearance (CrCl) calculator and Pediatric dosing (mg/kg/day) tools.
- **Micro-interactions:** Smooth, professional UI with glassmorphic elements and localized health data visualization.
- **Offline-Ready Data:** Uses a local clinical dataset for high-frequency search and critical dosing logic.

## 🛠 Technology Stack
- **Frontend:** Vanilla JS / HTML5 / CSS3 (Modern HSL system)
- **APIs:** openFDA API, NIH RxNav (RxNorm), NLM DailyMed
- **Design:** Custom clinical design system with deep blue accents and premium aesthetics.

## 🚀 Getting Started
1. **Clone the repo:**

   Open `index.html` in your browser (recommend serving with `npx serve .`).
